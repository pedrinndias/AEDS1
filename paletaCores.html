O Vocabulário da Cor

O tipo de dado que você visualiza dita a paleta que você deve usar. Escolher a paleta errada é como falar o idioma errado para os seus dados.

1. Paleta Categórica

Use para: Distinguir grupos discretos sem ordem (ex: tipos de produto, países).
Objetivo: Máxima distinção visual.

2. Paleta Sequencial

Use para: Dados numéricos que vão de baixo para alto (ex: receita, temperatura).
Objetivo: Mostrar magnitude.

3. Paleta Divergente

Use para: Dados com um ponto central significativo (ex: correlação, lucro/prejuízo).
Objetivo: Enfatizar desvios.

O Inimigo Oculto: A Paleta Arco-Íris

A paleta `jet` (arco-íris) é popular, mas desastrosa para a análise. Sua luminosidade não uniforme cria falsas fronteiras e esconde detalhes, distorcendo a percepção dos dados.

❌ Incorreto: Paleta Arco-Íris (Não Uniforme)

Os "saltos" de brilho entre ciano e amarelo criam a falsa impressão de uma mudança abrupta. A transição não é fiel à progressão linear dos dados.

✅ Correto: Paleta Perceptualmente Uniforme (Viridis)

A transição de luminosidade é suave e constante. Passos iguais nos dados correspondem a passos iguais na percepção da cor, garantindo uma representação honesta.

Visualizando a Importância de Features

A importância de uma feature é um dado quantitativo e ordenado. Uma paleta **sequencial** reforça essa hierarquia visualmente, onde a cor mais intensa representa a maior importância. Usar cores categóricas aqui seria um erro, pois implicaria que as features são apenas diferentes, não mais ou menos importantes.

Decodificando uma Matriz de Correlação

Uma matriz de correlação tem um ponto central (0) e valores que divergem para positivo (1) e negativo (-1). Uma paleta **divergente** é perfeita para isso. Ela usa cores distintas para correlações positivas e negativas, permitindo identificar instantaneamente a natureza e a força das relações entre as variáveis.
